# Samsung Web Clone - Gayatri Edition

A high-fidelity clone of the Samsung homepage built with React, focusing on modern UI/UX, responsive design, and premium typography.

## 🚀 Personal Updates
I have updated this project to include:
- **Custom Typography:** Integrated 'Samsung Sharp Sans' and 'Inter' font stacks for a cleaner look.
- **Modern UI Elements:** Added Glassmorphism to the navigation bar and "pill-shaped" buttons.
- **Interactivity:** Implemented smooth-scroll behavior and hover-scale effects on product cards.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Styling:** CSS3 (Custom Properties), Bootstrap, Material UI (MUI)
- **Icons:** Material Symbols / Lucide-React
